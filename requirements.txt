# WinClear - Windows Cleaner
# Зависимости для проекта

# Все используемые модули являются частью стандартной библиотеки Python
# Дополнительные зависимости не требуются

# Стандартные библиотеки:
# - os (работа с файловой системой)
# - shutil (операции с файлами и папками)
# - sys (системные функции)
# - time (работа со временем)
# - ctypes (взаимодействие с Windows API)
# - threading (многопоточность)
# - subprocess (выполнение команд)

# Минимальная версия Python: 3.6+
